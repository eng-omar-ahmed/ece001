Computer:

It is an electronic device programmed to process data and information and it performs four functions: input, processing, output and storage.

Computer components:

Hardware:

  • Internal components:(Motherboard, CPU, RAM, power supply, hard disk, screen card).

  • External components:(Keyboard, scanner, touch screen, digital camera, mouse, microphone, scanner, barcode reader, screen, printer, speaker, printer).

Software:

  • System software

  • Application software

Computer hardware:

Refers to physical computers that provide support for major functions such as input and processing (internal storage, calculation and control), output and secondary storage (for data and programs) and communications.

Computer software:

Known as programs or applications. They are classified into two categories: system software and application software.

input device: A piece of computer hardware used to enter data signals and control an information processing system such as a computer or information device.

output device:It is any piece of computer equipment that converts information into a human-readable form. It can be text, graphics, touch, sound and video.
