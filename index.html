Introduction to computers

COMPUTER SYSTEM:

A collection of entities (hardware, software, and live tools) designed to receive, process, manage and present information in a meaningful format.

Explanations;:

  • A computer is an electronic device that accepts data as inputs, and converts it under the influence of a set of special instructions called programs, to produce the required outputs (referred to as information).

  • The computer is described as an electronic device because; It consists of electronic components and uses electrical energy (such as electricity) to operate.

  • The computer has an internal memory that temporarily stores data and instructions awaiting processing, and even carries the intermediate result (information) before it is delivered to recipients via output devices.

  • The computer works on data using the instructions issued, which means that the computer cannot do any useful work on its own. It can only work according to the set of instructions issued.

  • The computer will accept data in one form and produce it in another. Usually data is kept inside the computer during processing.

Computer use areas:

  1. Fast word processing, editing and error correction.

  2. Dissemination of information widely.

  3. Drawing through drawing programs

  4. Solve complex equations.

  5. Produce images and graphics by converting data into illustrated forms

  6. Production and operation of video games.

Computer uses:

  1. Storing all information and data, especially digital and verbatim information, and carrying it back quickly.

  2. Simulation of some electronic devices such as video, phone, fax and TV.

  3. Control the stoppage of some machines and devices and control their operation.

  4. Used to enter global networks, for example, the Internet.

  5. Fast and accurate drawing of graphics and maps of any type and uses.
